# Churn Classification using Logistic Regression

## Goal
The goal of this project is to practice implementing a Logistic Regression classifier for customer churn.

## Description
This project was implemented as part of a public tutorial  Mr . Mirmirani for learning purposses.
A telecommunications company is concerned about the number of customers leaving their land-line business for cable competitors. They need to understand who is leaving.

## Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib
