# Hurt Attack Classification using SVM (Support Vector Machines)

## Goal

The goal of this project is practice implementing one of the classification methods for hurt attack prediction based on patient features.

## Description

The source of  project is Kaggle. I choose SVM method since there are 304 rows in the dataset..
 In addition, I checked  3 ways(linear, poly,and rbf) for kernel and linear has the best evaluation.
 Hence, I illustrate svm.SVC(kernel='linear') in the project.

## Evaluation

Confusion matrix
F1_score
Jaccard index

## Technologies

Python, Pandas, NumPy, Scikit-learn, Matplotlib
