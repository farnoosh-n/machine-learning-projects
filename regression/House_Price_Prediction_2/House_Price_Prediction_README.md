# House Price Prediction – Tehran

## Project Overview

This project predicts house prices in Tehran Regression models. The goal was to Improve R2-score.

## Dataset

Dataset: House Price Tehran (Kaggle source)

## Models

 * Simple Linear Regression
 * Mulitple Linear Regression
 * NoneLinearRegression (Polynomial regression)


## Evaluation

R² Score ≈ 0.41  
Metrics: MAE, MSE

## Technologies

Python, Pandas, NumPy, Scikit-learn, Matplotlib
