#  DBSCAN Clustering Using Density-Based

## Goal
The goal of this project is to practice implementing Dbscan clustering .

## Description
This project was implemented as part of a tutorial by  Mr. shirafcan and Mr. Mirmirani for learning purposses.
Two files are covered:
  *  Dbscan  Clustering on the Iris dataset 
  *  Dbscan Clustering on randomly generated data

## Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib
