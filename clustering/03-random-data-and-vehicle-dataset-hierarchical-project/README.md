#  Hierarchical Clustering Using Agglomerative

## Goal
The goal of this project is to practice implementing Hierarchical clustering and working with dendrograms.

## Description
This project was implemented as part of a public tutorial by Mr. Mirmirani for learning purposses.
Two examples are covered:
  * Hierarchical Clustering on randomly generated data
  *  Clustering the Vehicle dataset using both scipy and scikit-learn

## Technologies
Python, Pandas, NumPy, Scipy, Scikit-learn, Matplotlib
