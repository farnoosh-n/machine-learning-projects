# Customer Segmentation Using K-Means

## Goal
The goal of this project is to practice implementing K-Means clustering for customer segmentation.

## Description
This project was implemented as part of a public tutorial by Mr. Mirmirani for learning purposes.

Two examples are covered:
  * K-Means Clustering on a randomly generated dataset
  *  Using  K-Means for customer segmentation

## Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib
