# Customer Clustering 

## Goal
The goal of this project is practice implementing clustering methods on Customer  dataset.

## Description
The source of  project is Kaggle.
Three methods of clustering are used on the dataset :
  1- Hierarchical clustering
  2- K-means clustering
  3- DBSCAN (Density-based) clustering

## Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib
